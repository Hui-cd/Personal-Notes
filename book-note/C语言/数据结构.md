## 字符类型

- `char`: 通常用来存储字符。
  - **存储大小**: 1 字节
  - **值范围**: -128 到 127 或 0 到 255
  - **示例**: `char x = 'a';` // 存储字符 'a'
- `unsigned char`: 用于存储较大的字符值。
  - **存储大小**: 1 字节
  - **值范围**: 0 到 255
  - **示例**: `unsigned char y = 255;` // 存储最大的 unsigned char 值
- `signed char`: 明确指定为有符号。
  - **存储大小**: 1 字节
  - **值范围**: -128 到 127
  - **示例**: `signed char z = -128;` // 存储最小的 signed char 值

## 整数类型

- `int`: 最常用的整数类型。
  - **存储大小**: 2 或 4 字节
  - **值范围**: -32,768 到 32,767 或 -2,147,483,648 到 2,147,483,647
  - **示例**: `int a = -2147483648;` // 存储 int 的最小值
- `unsigned int`: 用于需要非负数的场景。
  - **存储大小**: 2 或 4 字节
  - **值范围**: 0 到 65,535 或 0 到 4,294,967,295
  - **示例**: `unsigned int b = 4294967295;` // 存储 unsigned int 的最大值

## 短整型

- `short`: 用于存储较小的整数。
  - **存储大小**: 2 字节
  - **值范围**: -32,768 到 32,767
  - **示例**: `short c = -32768;` // 存储 short 的最小值
- `unsigned short`: 存储非负的较小整数。
  - **存储大小**: 2 字节
  - **值范围**: 0 到 65,535
  - **示例**: `unsigned short d = 65535;` // 存储 unsigned short 的最大值

## 长整型

- `long`: 用于需要比 `int` 更大范围的整数。
  - **存储大小**: 4 字节
  - **值范围**: -2,147,483,648 到 2,147,483,647
  - **示例**: `long e = 2147483647;` // 存储 long 的最大值
- `unsigned long`: 用于存储较大的非负整数。
  - **存储大小**: 4 字节
  - **值范围**: 0 到 4,294,967,295
  - **示例**: `unsigned long f = 4294967295;` // 存储 unsigned long 的最大值
